# Урок 1: Агент делает первые шаги

## Добро пожаловать! @showdialog

Привет! Сегодня ты познакомишься с **Агентом** — твоим роботом в Minecraft.

Ты напишешь программу и агент будет слушаться твоих команд прямо из чата!

Нажми **Далее** чтобы начать.

## Шаг 1 — Кто такой агент?

**Агент** — это робот который живёт рядом с тобой в Minecraft.
Он ничего не делает сам — только выполняет твой код.

Сначала добавим самый простой блок: агент делает **один шаг вперёд**.

Найди ``||agent:agent move forward by 1||`` и добавь его в рабочее поле.

```blocks
// @highlight
agent.move(FORWARD, 1)
```

## Шаг 2 — Запускаем и смотрим

Нажми зелёную кнопку **Play** внизу редактора.

Посмотри на агента в игре — он сделал шаг вперёд!

Попробуй нажать Play ещё раз. Агент снова шагает.

```blocks
// @highlight
agent.move(FORWARD, 1)
```

## Шаг 3 — Управление из чата

Неудобно нажимать Play постоянно. Давай сделаем команду в чате!

Когда ты напишешь **"вперёд"** в чат Minecraft — агент сделает шаг.

Добавь ``||player:on chat command "вперёд"||`` и перемести блок движения **внутрь** него.

```blocks
player.onChat("вперёд", function () {
    // @highlight
    agent.move(FORWARD, 1)
})
```

## Шаг 4 — Больше шагов!

Одного шага мало. Измени число **1** на **5**.

Теперь по команде "вперёд" агент пройдёт сразу **5 блоков**!

```blocks
player.onChat("вперёд", function () {
    // @highlight
    agent.move(FORWARD, 5)
})
```

## Шаг 5 — Проверяем в игре

Нажми **Play**, зайди в Minecraft и напиши в чат: `вперёд`

Агент должен пройти 5 блоков вперёд!

```blocks
player.onChat("вперёд", function () {
    agent.move(FORWARD, 5)
})
```

## Отлично! @showdialog

Ты написал свою **первую программу** для агента!

**Что ты узнал:**
- Блок `agent move forward` двигает агента
- Команда `on chat command` запускает код из чата Minecraft

**Попробуй сам:**
- Измени `FORWARD` на `BACK` — агент пойдёт назад
- Измени число `5` на `10`
