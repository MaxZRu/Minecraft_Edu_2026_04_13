# Minecraft Education — Серия уроков по программированию агента

Добро пожаловать! Здесь 4 урока для обучения программированию в Minecraft Education Edition.

---

## Как открыть урок

1. Запусти **Minecraft Education Edition**
2. Войди в мир → нажми **C** (Code Builder)
3. Скопируй ссылку нужного урока и вставь в Code Builder

---

## Список уроков

### Урок 1 — Агент делает первые шаги `[Начинающий]`
Знакомство с агентом. Учимся двигать его вперёд командой из чата.

**Концепты:** `agent.move`, `player.onChat`

```
https://minecraft.makecode.com/#tutorial:github:MaxZRu/Minecraft_Edu_2026_04_13
```

### Урок 2 — Агент строит стену `[Начинающий]`
Агент ставит блоки и строит ровную стену из 6 блоков.

**Концепты:** `agent.setItem`, `agent.place`, `loops.repeat`

```
https://minecraft.makecode.com/#tutorial:github:MaxZRu/Minecraft_Edu_2026_04_13/tutorial-02-build-wall
```

---

### Урок 3 — Пульт управления агентом `[Средний]`
Создаём несколько команд сразу: вперёд, назад, лево, право, домой.

**Концепты:** несколько `onChat`, `agent.turn`, `teleportToPlayer`, параметры `num1`

```
https://minecraft.makecode.com/#tutorial:github:MaxZRu/Minecraft_Edu_2026_04_13/tutorial-03-chat-commands
```

---

### Урок 4 — Агент рисует квадрат `[Средний]`
Агент рисует квадрат из блоков на земле. Используем цикл внутри цикла.

**Концепты:** `agent.place DOWN`, вложенные циклы `repeat`

```
https://minecraft.makecode.com/#tutorial:github:MaxZRu/Minecraft_Edu_2026_04_13/tutorial-04-agent-square
```
